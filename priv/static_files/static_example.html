Static example from directory

This file is served from the priv/static_files directory because of the following route in cowboy_elixir_example.ex: 

         {"/static/[...]", :cowboy_static, {:priv_dir,  :cowboy_elixir_example, "static_files"}}

Static Assets

:priv_dir is also typically used to serve other static assets like images, stylesheets, javascript files, etc. The :priv_dir route matcher descends into subdirectories. This file includes several, explicitly listed below.

  • /static/images/sample.png
  • /static/css/styles.css
